  - 완료 조건: 설계 문서에 폴더 구조와 목록 원칙이 명시되어 있다.

- [x] Task 2. 확정 설정 분류 폴더 생성
  - 목표: `확정 설정/종족`, `확정 설정/지역`, `확정 설정/국가및단체` 폴더를 만든다.

  - 의존성: Task 1

- [x] Task 3. 분류별 목록 문서 작성

  - 완료 조건: 종족, 지역, 국가및단체 목록 문서가 모두 생성되고 대상이 빠짐없이 적혀 있다.
  - 의존성: Task 2

  - 목표: 생성된 분류 폴더와 목록이 설계와 일치하는지 확인하고 보고 문서를 남긴다.
  - 완료 조건: 검증 문서와 보고 문서가 작성되어 있다.
  - 의존성: Task 3
