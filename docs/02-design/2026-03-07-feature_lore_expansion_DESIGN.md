# PDCA Design: 신규 설정 - 미개척지 탐사 매뉴얼 (2026-03-07)

## 1. 개요
- **대상**: `1.개념설정/11.미개척지 탐사 매뉴얼.md` 생성
- **내용**: 미개척지 탐사의 5단계 절차 및 개척팀과 국가의 역할 분배 명시

## 2. 세부 설계 (서술 구조)
마스터가 제공한 5단계를 바탕으로 다음과 같이 상세화하여 서술합니다.

1. **[제1장] 탐사의 서막: 다중 운용 체계**
   - 탐사 효율과 생존율을 높이기 위해 소수가 아닌 다수의 전문 인력이 투입되는 이유 서술.
   - 각 직업군(수호자, 마법사 등)의 유기적인 협력을 강조.

2. **[제2장] 안개의 위협: 마경의 심장부로**
   - '마경의 안개'가 단순한 자연현상이 아닌 흑마법적 간섭임을 서술.
   - 안개 속에서 벌어지는 기만과 환각, 그에 대응하는 탐사팀의 경계 태세.

3. **[제3장] 토템과 수호자: 원인 제거**
   - 안개를 생성하고 유지하는 근원인 '토템'의 구조.
   - 토템을 지키는 '수호자'와의 전투 양상과 전략적 중요성.

4. **[제4장] 포탈의 확보: 보급로의 완성**
   - 토템 근처에 반드시 존재하는 고대 포탈의 발견과 활성화.
   - 포탈이 확보됨으로써 탐사팀의 역할이 마무리됨을 강조.

5. **[제5장] 정화와 증축: 국가적 과업**
   - 영역 확보 후 진행되는 단계별 정화 작업(성수, 마법 진동 등).
   - 영구적인 개척지로 만들기 위한 외벽 증축과 국가 행정력의 투입.

## 3. 역할 분담 명시
- **개척팀(Pioneer Team)**: 1~4단계(투입, 탐사, 전투, 거점 확보) 전담. 고위험 고수익의 현장 실무.
- **국가(State Administration)**: 5단계(정화, 건설, 유지) 전담. 안정적인 영토 확장과 행정 지원.
