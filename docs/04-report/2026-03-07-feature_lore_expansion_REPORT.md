1. **탐사 프로세스 정립**:
   - 다중 투입 → 안개 돌파 → 토템 파괴 → 포탈 확보 → 정화/증축으로 이어지는 체계적인 5단계 개척 절차 수립.

3. **세계관 연계**:
   - 기존의 '마경의 안개', '토템', '고대의 포탈' 설정을 하나로 묶어주는 실전 지침서 역할을 수행.
