# Plan: 중앙 문명 및 신앙 체계 로어 업데이트

## 1. 개요 (Overview)
- **대상**: `1.개념설정\6.중앙 문명과 외곽의 마경.md`, `1.개념설정\8.신앙의 계보와 신격의 원리.md`
- **배경**: 마스터의 최신 설정(다국가 체제, 신격의 존재 원리)을 반영하여 세계관의 세부 구조를 더욱 정교하게 다듬음.

## 2. 목표 (Goals)
- 중앙 문명을 단일 체제가 아닌 복수 국가의 갈등/협력 구조로 재정립.
- 신격의 본질을 '개념의 이미지화'와 '신앙 의존적 생존'으로 명확히 규정.
- 디바인 보상이 '자신의 믿음의 회귀'임을 서술.
- 전 문서에서 '에코 게네시스' 용어 완전 배제.

## 3. 상세 태스크 (Tasks)
- [ ] [Plan] 신규 로어 반영 계획 수립 (@2026-03-08)
- [ ] [Design] 각 문서별 수정 섹션 및 핵심 문장 설계
- [ ] [Do] '중앙 문명' 문서 업데이트 (다국가 체제 반영)
- [ ] [Do] '신앙의 계보' 문서 업데이트 (신격의 원리 전면 수정)
- [ ] [Check] '에코 게네시스' 용어 잔존 여부 전수 검사 및 제거
