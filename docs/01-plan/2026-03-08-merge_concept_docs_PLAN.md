# Plan: 개념 설정 문서 1, 2번 병합

## 1. 개요 (Overview)
- **대상**: `1.개념설정\1.무한의 땅.md`, `1.개념설정\2.넘어온 자들.md`
- **배경**: 세계의 물리적 배경(무한의 땅)과 거주민의 기원(넘어온 자들)은 세계관의 '시초'라는 점에서 긴밀한 관계를 가짐. 이를 하나의 문서로 통합하여 서술의 일관성과 검색 편의성을 확보함.

## 2. 목표 (Goals)
- 물리적 배경과 인구학적 배경을 유기적으로 연결.
- 중복된 내용을 정리하고 서술의 흐름을 개선.
- 옵시디언의 [[링크]] 체계를 유지하여 문서 간 연결성을 강화.

## 3. 범위 (Scope)
- **병합 결과물**: `1.개념설정\1.세계의 기원과 인류의 정착.md` (가칭)
- **삭제 예정**: 기존 1, 2번 문서.
- **참조 업데이트**: 1, 2번 문서를 참조하는 다른 문서들의 링크 업데이트 필요 여부 확인.

## 4. 전략 (Strategy)
1. **문서 구조화**: '루테라의 물리적 특성' → '주인 없는 땅의 철학' → '넘어온 자들의 도착과 연합' 순으로 구성.
2. **연결성 강화**: 물리적 배경인 '무한의 땅'이 '넘어온 자들'의 정착 방식에 미친 영향을 강조하는 문장 추가.
3. **용어 통일**: '루테라' 및 '넘어온 자들'에 대한 서술을 일관되게 정렬.
