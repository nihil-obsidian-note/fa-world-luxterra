# 문서 작성 및 관리 규칙

## 1. 파일 명명 규칙
- 폴더명 앞에는 정렬을 위해 숫자를 포함합니다. (예: `1.개념설정`)
- 파일명은 명확한 주제를 나타내야 합니다. (예: `1.세계의 기원과 인류의 정착.md`)

## 2. 문서 구조
- 모든 문서는 최상위 제목(`#`)을 하나만 포함합니다.
- 하위 섹션은 `##`, `###`을 순차적으로 사용합니다.
- 핵심 용어는 `[[링크]]` 형태의 위키 링크를 사용합니다.

## 3. 메타데이터 (YAML Frontmatter)
- 문서 상단에 작성일, 수정일, 태그를 포함하는 것을 지향합니다.
## 4. 금지어 및 용어 제한
- **에코 게네시스 (Echo Genesis)**: 본 세계관에서 더 이상 사용하지 않는 용어입니다. 발견 시 삭제하거나 적절한 대체 용어를 검토합니다.
